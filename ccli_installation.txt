1. install expect

$ apt-get install expect

2. install Culberson's colouring programs

$ cd /opt
$ mkdir color
$ cd color
$ wget --no-check-certificate http://www.cs.ualberta.ca/%7Ejoe/Coloring/Colorsrc/color.tar.gz
$ tar xzvf color.tar.gz
$ mkdir bin
$ make all

3. install docopts

$ cd /opt
$ git clone https://github.com/docopt/docopts.git
$ cd docopts
$ ./get_docopts.sh
$ cp docopts /usr/local/bin

4. install ccli

$ cd /opt
$ git clone https://gitlab.com/mjhgraph/ccli.git (use github instead, or download link)
